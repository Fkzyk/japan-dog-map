### STEP 3: admin.html を改修

**変更1: load処理をOAuthセッションチェックに置き換え（行346〜351）**
```javascript
window.addEventListener('load', function() {
  sb = window.supabase.createClient(SUPABASE_URL, SUPABASE_ANON_KEY);
  sb.auth.getSession().then(function(result) {
    if (!result.data.session) {
      window.location.href = '/admin-login.html'; return;
    }
    sb.rpc('is_admin').then(function(r) {
      if (r.error || !r.data) {
        alert('管理者権限がありません。');
        sb.auth.signOut().then(function() { window.location.href = '/admin-login.html'; });
        return;
      }
      showAdminSection(); loadSubmissions(); loadRequestsBadge();
    });
  });
});
```

**変更2: logout関数（行375〜382）**
```javascript
function logout() {
  sb.auth.signOut().then(function() {
    window.location.href = '/admin-login.html';
  });
}
```

**変更3: 全RPC呼び出しを v2 に切替（input_password 削除）**

| 変更前 | 変更後 |
|---|---|
| `admin_get_submissions` + `{input_password: adminPassword}` | `admin_get_submissions_v2` + `{}` |
| `admin_update_status` + `{input_password, submission_id, new_status}` | `admin_update_status_v2` + `{submission_id, new_status}` |
| `admin_update_submission` + `{input_password, submission_id, ...}` | `admin_update_submission_v2` + `{submission_id, ...}` |
| `admin_delete_submission` + `{input_password, submission_id}` | `admin_delete_submission_v2` + `{submission_id}` |
| `admin_get_reviews` + `{input_password}` | `admin_get_reviews_v2` + `{}` |
| `admin_update_review_status` + `{input_password, review_id, new_status}` | `admin_update_review_status_v2` + `{review_id, new_status}` |
| `admin_get_deletion_requests` + `{input_password}` | `admin_get_deletion_requests_v2` + `{}` |
| `admin_update_request` + `{input_password, request_id, new_status}` | `admin_update_request_v2` + `{request_id, new_status}` |
| `admin_get_overrides` + `{input_password}` | `admin_get_overrides_v2` + `{}` |
| `admin_upsert_override` + `{input_password, p_spot_id, ...}` | `admin_upsert_override_v2` + `{p_spot_id, ...}` |
| `admin_delete_override` + `{input_password, p_spot_id}` | `admin_delete_override_v2` + `{p_spot_id}` |
